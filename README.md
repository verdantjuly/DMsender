구글 스프레드시트 - 슬랙 연동 자동화 앱 스크립트 코드

1. 구글 스프레드 시트 2열에 이메일을 입력한다.
2. 해당 이메일로 워크스페이스에 가입된 슬랙 유저에게 DM이 간다.

- 제작 목적<br/>
스파르타 코딩클럽 커리어톤 업무 중 자동화의 필요성과 가능성을 느껴 제작하였습니다.
